# 手机推送配置指南（只配一次，之后每天自动）

目标：每天定点手机锁屏弹一条「📰 今日情报站日报已就绪」，点开直接看云端日报。

- **iPhone** → 用 Bark（第一步）
- **安卓** → 用 ntfy（见下方「安卓/通用：ntfy」一节，安卓装不了 Bark）
- 支持任意多人混合，每人各自装好把自己的 key/topic 发给你即可。

---

## 安卓 / 通用：ntfy（安卓必看，约 5 分钟）

ntfy 是 Bark 的安卓对等品，免费、无需注册、点开能跳网页。iPhone 也能用。

1. 手机应用商店搜 **ntfy**（安卓 Google Play / F-Droid，iOS App Store），安装。
2. 打开 App，点 **+** 订阅一个 **topic**。topic 名字自己起，但要**难猜**（它相当于密码），例如 `jimiding-intel-9x7k2`。
3. 把这个 topic 名告诉配置的人。
4. 填法二选一：
   - 本地：`push_config.json` 的 `ntfy.topics` 填进数组，如 `["jimiding-intel-9x7k2"]`
   - 云端：GitHub 仓库 Secret 新建 **`NTFY_TOPICS`**，多个 topic 用逗号隔开。
5. 多个安卓用户：订阅**同一个** topic 就都能收到；想各自独立就各填各的 topic（逗号隔开）。

> 验证：电脑跑 `NTFY_TOPICS=你的topic python notify.py`，手机几秒内应收到横幅。

---

## 第一步：拿 Bark Key（iPhone，约 5 分钟）

1. iPhone App Store 搜 **Bark**，安装（免费、开源、无广告）。
2. 打开 App，首页会显示一条你的专属 URL，形如：
   `https://api.day.app/AbCdEf123456/...`
3. 复制中间那段 **`AbCdEf123456`**（你的 key）。
4. 填进 `push_config.json` 的 `bark.key`。

> 验证：填好后在电脑跑 `python notify.py`，iPhone 应立刻收到一条横幅，就成了。

---

## 第二步：建 GitHub Pages（约 10 分钟，让手机能点开网页）

1. 登录 GitHub，新建一个仓库（比如叫 `intel-daily`），可设为 Public。
2. 进仓库 **Settings → Pages**：Source 选 `Deploy from a branch`，Branch 选 `main` / `(root)`，Save。
3. 几分钟后 Pages 地址生效，形如 `https://你的用户名.github.io/intel-daily/`。
4. 填进 `push_config.json`：
   - `github_pages.repo` = `你的用户名/intel-daily`
   - `github_pages.page_url` = `https://你的用户名.github.io/intel-daily/today.html`
5. 确保本机 git 能推这个仓库：终端跑一次 `gh auth login`（推荐），或配好 PAT/SSH。

> 验证：电脑先跑 `python publish.py` 生成当天快照，再跑 `python deploy.py`，
> 然后手机浏览器打开你的 page_url，应能看到今天的日报。

---

## 配好之后

什么都不用做。每天 9:00 定时任务会自动：
抓取 → 降噪 → 归档 → **推 GitHub Pages → Bark 弹手机**。
你早上锁屏收到横幅，点开就是当天日报。

## 想换成微信推送？

把 `notify.py` 里拼 Bark URL 那段，换成 Server酱/PushPlus 的接口即可，其余链路不变。
